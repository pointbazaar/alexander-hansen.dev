List of non-mainstream Programming Language Projects

These days there are many Programming Languages out there.
It is very popular for people interested in Compilers to create their own Languages.
This is just a list of some interesting, not so well known ones i have found.
If you are actively developing a Programming Language, send me an email to have it added to the List.
