Dragon : my first Language Project

I tried to create a Programming Language and got a working Compiler that could compile a Hello World Program.
Also created a tiny Math Library for it with very basic subroutines.
You can see the Project on my Github.

The Project failed in being a Programming Language, but i learned many things from it.
As the meme goes, most Programming Languages fail.

In my Opinion, Dragon did not gain any adoption because it was worse in every way compared to most popular Programming Languages.

But i learned about some interesting ideas like Incremental Compilation , Dependent Types and Java Bytecode in the Process.

In the future i want to focus more on smaller, simpler Projects related to Programming Languages and Compilers.
Maybe to contribute to an existing Language or to create simple tools that improve a language ecosystem or to improve tooling developers in general.

Ideas for future projects:

  • comcat : like 'cat', but it just filters out the comments from source files for many languages
  • comcat : like 'cat', but it just filters out the dependencies from source files for many languages
  • strcat : like 'cat', but it just filters out the strings from source files for many languages
  • A source code generator that generates source code in some language based on a model, specification or MySQL database schema
